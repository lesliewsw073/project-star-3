# 开局流程：取名、公司名与三选一签约

> **定案要点**
> - 三选一 UI 用 **行动名**，不用艺人简历标签
> - 三人 **流程结构相同**（选行动 → 该角色专属签约剧情 → 后续节点），**正文各自独立**
> - 命名／公司名在选艺人 **之前** 完成并锁定

---

## STAGE 1 · 世界观引入

> 漆黑屏幕，打字机声，红字逐行。不搞宏大叙事，直奔主题。

**旁白（定案）**

- 「你不是什么怀揣梦想的救世主。你只是一个倒霉的接盘侠——前任老板卷款跑路，留下了这间随时会被断水断电的破办公室，以及一屁股烂账。」
- 「想活命？先在清算协议上签了你的卖身契。」

---

## STAGE 2 · 取名与公司名（`OpeningProfileDialog`）

> **程序**：弹出 UI，玩家填写后 **立刻锁定**，不可再改。  
> **锁定字段**：`ProtagonistManager.profile_locked`、`PlayerManager.company_name_locked`

| 输入项 | 变量／用途 |
|--------|------------|
| 主角姓 + 名 | `protagonist_full_name`、`{player_address}` 等对话变量 |
| 主角生日 | `protagonist_birthday`（`MM-DD`）；写入存档 · 周会 `protagonist_birthday` 节日触发（节日前例会）· **UI 待实作** |
| 公司名称 | `{company_name}`；状态栏、标题、对话均已接入 |

**剧情包装**

- 秘书（`secretary`）推过《协议》与钢笔
- **secretary**：「别发呆了，催款电话都快把前台打爆了。在这签字吧，老板。」
- 命名完成后 **secretary**：「很好，`{protagonist_full_name}`。从今天起，`{company_name}` 的所有债务正式移交到你名下。账上剩下的流动资金，也就够给保洁阿姨发这个月的薪水。」

---

## STAGE 3 · 三选一（行动，非简历）

> **程序标签**：`select_opening_action_panel`（占位名）  
> **UI 原则**：三个选项只显示 **玩家行动**，不显示艺人姓名、简历、标签。

| 选项 | UI 文案（定案） | 对应艺人 |
|------|-----------------|----------|
| 1 | **下楼透透气** | `artist_001` 夏知秋 Yoko |
| 2 | **去看场舞台剧** | `artist_002` Valeria |
| 3 | **打开电视看看** | `artist_003` 米语 |

选中后进入该线 **专属发现剧情**，再进入该线 **专属签约剧情**；三人互不共用正文。

---

## 选项 1 · 下楼透透气 → artist_001

### 发现（路边弹唱）

- 玩家下楼透气，在路边听见 **001 自弹自唱英伦摇滚原创**
- 体现人设：有勇气（没舞台也敢唱）、原创热情、大白嗓不自知（签约／前期再点破）

### 签约剧情

- Event 正文：[[02_Story_Events/1_Main_Story/Artists/artist_001/00_street_sign_01]]
- **001 签约特点**：路边相遇 + 名片；**Gameplay 跳过** 父母商量／回公司签字（台词里可提「跟父母商量」，但不另做 event）
- 与 002 **带回家细看 + 电话略过**、003 **邀来公司签约** 形成三种不同体验

### 签约后 · 时间跳跃 → 周日第一次例会

- **001 线不做** 003 式「签约当日 office 首日」
- 签约剧情 **结尾加时间跳跃旁白**，示例：
  - *「数日过去。第一个周日，你第一次以老板身份坐在会议桌前。」*
- 接着播放 **001 专属首次例会**：[[02_Story_Events/1_Main_Story/Artists/artist_001/01_first_meeting_01]] · `story_meeting_artist_001_first_session_01`

```text
取名/公司名 → 选「下楼透透气」→ 00_street_sign_01 → 【旁白跳日】→ 01_first_meeting_01
```

## 选项 2 · 去看场舞台剧 → artist_002

### 发现（台上极亮、台下极缩）

- 玩家去看舞台剧；**不宜当街搭讪式发现**
- **台上**：002 天赋极高、虽然是配角，可是登场表演震撼人心
- **台下**：有点畏缩，迅速低着头回到自己的保护壳
- 主角因「反差」记住她，再安排接触／邀约

### 签约剧情

- Event 正文：[[02_Story_Events/1_Main_Story/Artists/artist_002/00_theater_sign_01]]
- **002 签约特点**：剧院散场递合同，贝尔**带回家细看**；条款与行程 **电话沟通略过** → 周日例会正式落地
- 与 001 **名片事后商量**、003 **邀来公司签约** 形成三种不同体验

### 签约后 · 时间跳跃 → 周日第一次例会

- **002 线不做** 003 式「签约当日 office 首日」
- 签约剧情 **结尾加时间跳跃旁白**，链至 **002 专属首次例会**：[[02_Story_Events/1_Main_Story/Artists/artist_002/01_first_meeting_01]] · `story_meeting_artist_002_first_session_01`

```text
取名/公司名 → 选「去看场舞台剧」→ 00_theater_sign_01 → 【旁白跳日】→ 01_first_meeting_01
```

## 选项 3 · 打开电视看看 → artist_003

### 前置说明（新增 · 短）

1. 玩家选择「打开电视看看」
2. **旁白（1～2 屏）**：
   - 电视正在播放一场体育赛事；女主播（米语）的形象让人眼前一亮
   - 主角立刻让秘书打电话给电视台，询问联系方式，让其发起邀约
3. 接着屏幕变黑，显示隔了一天。

### 衔接既有 003 链

- 马上切到 **敲门 + CG**（`sign_knock_office`）→ 已有签约／office 正文：
  - [[02_Story_Events/1_Main_Story/Artists/artist_003/00_office_sign_01|00_office_sign_01]]
- 再进 **003 专属首次周会**：[[02_Story_Events/1_Main_Story/Artists/artist_003/01_first_meeting_01]] · `story_meeting_artist_003_first_session_01`

```text
取名/公司名 → 选「打开电视看看」→ 旁白+电话邀约（短）→ 敲门/CG → 00_office_sign_01 → 01_first_meeting_01
```
