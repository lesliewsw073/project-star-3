# 大地图偶遇 · 台词库台账

> **路径**：`2_Daily_Loops/Artists/Map_Encounters/`  
> **写前必读**：[[01_Characters/人设集合_大地图随机对话]]  
> **程序备忘**：回代码时对照 `StoryTriggerManager` 与行程/压力/失败率数据源，逐项验收 `trigger_tier` / `selection_priority` / 互斥。

---

## 目录层级（定案）

```text
Map_Encounters/
├── 台词库台账.md
└── artist_NNN/
    ├── Permanent/       # A · 普通常驻 · trigger_tier: permanent
    ├── Flag_Unlock/     # B · CG/主线解锁后追加常驻 · trigger_tier: flag_unlock
    └── Conditional/     # C · 条件/限时 · trigger_tier: conditional
```

| 子夹 | trigger_tier | selection_priority（建议） | 入池条件 |
|------|--------------|---------------------------|----------|
| `Permanent/` | `permanent` | **0**（兜底） | 入队后即可抽 |
| `Flag_Unlock/` | `flag_unlock` | **10**（与 A 同级随机，或略加权） | `required_flags` 已满足 |
| `Conditional/` | `conditional` | **100** 限时 · **50** 状态 | 见下表；**同 tier 内**再比 `selection_priority` |

### 程序抽选顺序（剧本定案 · 代码待实现）

1. 收集当前艺人 **全部 eligible** event（flags + 条件公式为真）
2. 取 **最高** `selection_priority` 的 tier；若多条，**同 priority 内** `pool_variant` 随机
3. **互斥示例**：`major_job_completed_recent`（100）> `weekly_fail_rate_above_50`（50）> `Permanent`/`Flag_Unlock`（0～10）
4. C 类 **`expires_after_days`** 过期即从池移除
