# 周会聊天 · 台词库台账

> **路径**：`2_Daily_Loops/Artists/Meeting_Weekly/`  
> **写前必读**：本页 + [[00_Project_Spec/剧本event正文格式标准]] §六 · Meeting_Weekly  
> **程序备忘**：回代码时对照 `StoryTriggerManager` 验收 `meeting_tier` / `selection_priority` / 节日窗口 / 每周一次互斥。

---

## UI 与互斥（定案 · 2026-06-19）

| 规则 | 说明 |
|------|------|
| 每周必有例会 | 周日进入会议；点艺人 → 弹「聊天」选项 |
| **每艺人每周仅 1 次** | 三选一（或四选一含主线）中 **只可选 1 条**；选后本周该艺人聊天按钮 **变灰** |
| 选项数量 | 默认 **3 个日常**；若有一次性主线 meeting → **+1** → 共 4 个 |
| 选项来源 | 当周 **eligible** 集合按 `selection_priority` 取 **Top3** 进弹窗（主线 meeting 另插，不占日常 3 格） |
| 结算 flag | 建议 `meeting_chat_used_{artist_id}_week_{year}_{week}` |

**不加进周会池**：公司声望、压力／羁绊档／大通告单次／003 去留预警／未列节日等。

## 目录层级（定案）

```text
Meeting_Weekly/
├── 台词库台账.md
└── artist_NNN/
    ├── Award/                 # 奖项 · meeting_tier: award
    ├── Festival/              # 节日 · meeting_tier: festival
    ├── Original_Song/         # 原创汇报 · meeting_tier: original_song
    ├── Schedule_Performance/  # 通告完成率 · meeting_tier: schedule_performance
    └── Basic/                 # 基础闲聊 · meeting_tier: basic
```

### 优先级（高 → 低）

| meeting_tier | selection_priority | 说明 |
|--------------|-------------------|------|
| `award` | **100** | 候选 / 拿奖 / 落选 · 各状态 **1 条固定**；条件满足 **必进当周选项池** |
| `festival` | **80** | 见下表 · 仅 **节日前** 的例会周 |
| `original_song` | **60** | 对白骨架固定 · 歌名／剧本片段 **读参数** |
| `schedule_performance` | **40** | 近期通告完成率（阈值待代码定） |
| `basic` | **0** | 问过往、目标、书、风格等；题类似、三人答不同 |

同 tier 内多条 eligible → 同 priority 内 `pool_variant` 随机；但进弹窗仍只展示 **Top3 日常**（不含主线 +1）。

| `protagonist_birthday` | 主角生日 | 读档／开局建档 | 玩家设定 |

### 节日触发窗口

- **只在节日前** 的那次 **周日例会** 进 eligible 池  
- **节日过后** 的周日例会 **不再** 触发该节日对白  

例：圣诞 12/25 週三 → 12/22 例会 ✅；12/29 例会 ❌。  
艺人生日、主角生日：同一规则（生日 **之前** 的例会周）。

每人每个 `festival_key`：**1 条固定对白**（待写）。
