# 游戏开局流程与世界观初始化 (start_flow.md)

## 1. 玩家/公司命名阶段
* **设计原则**：不搞煽情的“拯救娱乐圈”口号。玩家扮演的是一个被推上鸭子架的接盘侠。
* **流程**：
    1. 昏暗的办公室，桌上是一叠催款单。
    2. 秘书金妮把一份《法人变更协议》砸在桌上，冷淡地提醒你签名。
    3. **【玩家命名】**：在协议的“新法人代表”一栏输入玩家名字。
    4. **【公司命名】**：在协议的“破产重组企业名称”一栏输入公司名字。
* **金妮的反应**：
    * 名字签完后，金妮会冷笑着把协议收走：“从现在起，XX（公司名）的债务就全部由你 XX（玩家名）承担了。祝我们死得太难看。”

## 2. 核心状态初始化
* **初始资金**：极其紧张（仅够维持基础运营和第一个月的最低基本工资）。
* **初始资产**：一间随时可能被物业断水断电的旧写字楼，一个随时准备看你笑话的秘书。
* **核心目标**：在 6 个月内把公司从破产边缘拉回来，否则直接进入“天台见”垃圾结局。
